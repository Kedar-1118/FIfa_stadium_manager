# Command Center: Canvas Engine

## Why it exists
Configures WebGL rendering of geospatial elements and 3D stadium structures.

## Responsibility
Overlays Deck.gl layers over maps and draws 3D geometries.

## What classes/files belong here
- MapCanvas.tsx (initializes base Mapbox layouts)
- ThreeStadiumModel.tsx (loads and draws internal stadium meshes)
- HeatmapLayer.ts (defines crowd flow and density shaders)
