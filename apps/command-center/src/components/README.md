# Command Center: Components

## Why it exists
Sleek visual dashboard blocks following premium styling principles.

## Responsibility
Displays interactive metrics tables, charts, navigation headers, sidebars, and alert widgets.

## What classes/files belong here
- AlertBanner.tsx (real-time notification slide-ins)
- GateStatusTable.tsx (flow metrics display grid)
- ControlPanel.tsx (buttons allowing manual overrides for gates and digital signage)
