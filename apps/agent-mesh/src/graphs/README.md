# Agent Mesh: Graph Orchestration

## Why it exists
Defines LangGraph routing meshes, state transitions, and supervisor behaviors.

## Responsibility
Implements conditional logic, routing state paths, and human-in-the-loop validation checkpoints.

## What classes/files belong here
- SupervisorGraph.ts (coordinates task delegations among specialists)
- EmergencyEscalationGraph.ts (handles critical safety notifications)
