# Agent Mesh: Specialized Agents

## Why it exists
Individual agent profiles with dedicated prompts, tools, and reasoning boundaries.

## Responsibility
Constructs agentic prompts, calls specific domain tools, and reports resolutions to the supervisor.

## What classes/files belong here
- CrowdDynamicsAgent.ts (handles bottlenecks and rerouting layouts)
- SecurityMedicalAgent.ts (manages dispatches and emergency responses)
- TransportGateAgent.ts (coordinates gates and public transport integration)
- SustainabilityAgent.ts (monitors waste and utility systems)
