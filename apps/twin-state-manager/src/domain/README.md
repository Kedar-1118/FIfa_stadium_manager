# Twin State Manager: Domain Layer

## Why it exists
Core model of the physical stadium twin state.

## Responsibility
Maintains spatial and numerical boundaries, handles capacity checks, and processes flow metrics mathematically.

## What classes/files belong here
- StadiumSector (defines bounds and current capacity indicators)
- GateState (represents turnstile status and flow stats)
- TelemetryMetric (encapsulates real-time sensor events)
