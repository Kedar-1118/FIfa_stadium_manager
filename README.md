# StadiumOS AI Root

## Why it exists
Root directory of the StadiumOS AI project, coordinates microservices, agentic mesh, and frontends in a clean monorepo.

## Responsibility
Manages project-wide configuration, dependencies, shared workspaces, build scripts, and local developer environment orchestration.

## What classes/files belong here
- package.json (pnpm workspace configurations)
- go.work (Go microservices workspace coordination)
- .gitignore (standard repository patterns to exclude)
- stadium_os_architecture.md (high-level system design)
